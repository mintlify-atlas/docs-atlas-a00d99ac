## Tech Stack

- **Frontend**: React 19.2 with modern hooks and component architecture
- **Build Tool**: Vite 7.2.4 for lightning-fast development and optimized builds
- **Styling**: Vanilla CSS with CSS variables, Flexbox, and Grid
- **Backend API**: Node.js serverless functions with Nodemailer
- **Security**: Google reCAPTCHA v2 for form protection
- **Icons**: Lucide React for scalable vector icons
- **Linting**: ESLint 9.39.1 with React-specific rules
